### [Server/API] — PR #24834: server: (router) move model downloading to dedicated process

**Merged:** 2026-06-22 | **Author:** @ngxson

**Summary:** Refactors the server's router mode to spawn a dedicated child process for model downloading instead of using an in-process thread, making the download handling consistent with how model execution is managed. **Affected areas:** server, router mode, model download, process management
