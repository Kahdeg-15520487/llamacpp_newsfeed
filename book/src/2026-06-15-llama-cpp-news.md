### [Server/API] — PR #24080: UI/svg block rendering

**Merged:** 2026-06-15 | **Author:** @ServeurpersoCom

**Summary:** Adds inline rendering of fenced ```svg code blocks generated by models in the chat UI, reusing the existing mermaid diagram UX (block wrapper, header, copy/preview buttons, and a zoom/pan/download preview dialog). ### [Bug Fix] — PR #24621: convert: fix lora base model arch retrieval

**Merged:** 2026-06-14 | **Author:** @CISC

**Summary:** Fixes a crash in `convert_hf_to_gguf.py` when converting LoRA adapter weights for certain model architectures. The issue occurred because the converter was trying to resolve the base model's architecture by reading `hparams` directly, which could fail for models with non-standard parameter layouts.
